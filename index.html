HTML Tags & CSS Properties Reference

HTML Tags
Tag Name Description
<strong> Strong This will generally render a text with bold emphasis
<ul> Unordered List Refers to an unordered list of items, rendered as a bulleted list
<ol> Ordered List refers to an ordered list of items, rendered as a numbered list
<p> Paragraph The structural grouping of related contents such as block of texts
<img> Image This embeds an image into a document
<figcaption> Figure Caption This is a child element to <figure> describing the content of its parent element
CSS Properties
Property Value Description
letter-spacing normal This allows user to alter the space between characters in other to justify text
position absolute This removes the element from the normal document flow and no space is created for the element in the page layout
position static This positions the element according to the normal flow of the document
text-align center This centers the inline contents within the line box
text-transform uppercase This converts all the characters to uppercase
overflow scroll This clips overflow content at the element's padding box and overflow content can be scrolle into view using scroll bars
